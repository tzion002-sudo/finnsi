**תאריך:** 2026-05-26  
**גרסה:** V2.9.4  
**סטטוס:** מאושר — ממתין ליישום

---

## מטרה

כאשר המשתמש מקבל דוח רבעוני מחברת ביטוח/פנסיה, הוא יעלה אותו לאפליקציה. המערכת תחלץ אוטומטית את כל הנתונים (יתרה, תשואה, הפקדות, דמי ניהול), תעדכן את הקופה המתאימה, ותשמור נקודת היסטוריה לגרפים.

**משתמשים:** ציון (6 קופות), זיו (3 קופות), הראל, ליאם — כל בני המשפחה.  
**סוגי קופות:** פנסיה, גמל, השתלמות, חיסכון לכל ילד.  
**חברות נתמכות:** מנורה מבטחים, מיטב, כלל, אלטשולר שחם, הראל, מגדל, פסגות, הפניקס, ילין לפידות.

---

## ארכיטקטורה — זרימת נתונים

```
משתמש מעלה PDF (מ"מחסן דוחות")
        ↓
pdfParser.js — extractTextFromPDF() → parsePDF()
        ↓
מחלץ: institution, reportType, owner, reportDate,
       balance, ytdReturn, deposited, fees, investmentTrack
        ↓
matchFundToAsset() — מתאים לפי owner + fundType + institution
        ↓
אם התאמה ברורה:
  • כותב ל-fund_history/{autoId}   ← נקודת ציר זמן
  • מעדכן assets (יתרה נוכחית + תאריך)
  • שומר רשומת ארכיון ב"מחסן דוחות"

אם לא ברור → דיאלוג בחירה → המשתמש בוחר קופה → ממשיך
```

## Firestore — מבנה נתונים חדש

### קולקציה: `fund_history`

כל מסמך = דוח רבעוני אחד:

```javascript
{
  owner:           "ציון",                    // שם בעל הקופה
  fundType:        "pension",                 // pension | study_fund | gemel | children
  institution:     "מנורה מבטחים",
  reportDate:      "2026-03-31",              // ISO — תאריך הדוח
  balance:         657178,                    // יתרה נוכחית ₪
  ytdReturn:       -4.04,                     // תשואה מתחילת שנה %
  deposited:       15844,                     // הפקדות ברבעון ₪
  fees:            360,                       // דמי ניהול ברבעון ₪
  investmentTrack: "מחקה S&P 500",
  feeFromBalance:  0.11,                      // דמי ניהול מצבירה %
  feeFromDeposit:  1.39,                      // דמי ניהול מהפקדה %
  uploadedAt:      "2026-05-26T10:00:00Z",
  fileName:        "March 0 (1).pdf",
  assetId:         "asset_abc123",            // קישור ל-assets
}
```

**מניעת כפילות:** לפני כתיבה, בדיקה לפי `owner + fundType + institution + reportDate`. אם קיים — שאלה למשתמש.

### 3. `src/Dashboard.jsx` — שינויים

#### DocumentsTab (מחסן דוחות)
- מחליף `file.text()` בקריאה ל-`parsePDF()` האמיתי
- מסיר כפתור "החל" ידני — הכל אוטומטי
- מוסיף שורת סטטוס לכל קובץ: `"מנורה מבטחים · פנסיה · ציון · ₪657,178 ✅"`
- מוסיף דיאלוג בחירה כשהתאמה לא חד-משמעית

#### SavingsTab (חסכונות) — שדרוג
מוסיף **מעל** הכרטיסים הקיימים:

**1. גרף מסכם (BarChart / LineChart)**
- ציר X: רבעונים (Q1 2025, Q2 2025...)
- ציר Y: יתרה כוללת
- פילטר: כולם / ציון / זיו / הראל / ליאם
- מקור: `fund_history` (מסונן לפי owner)

**2. כרטיס קופה מורחב** (לאחר לחיצה על קופה קיימת)
```

[גרף קטן: Q1 2025 → Q2 2025 → Q3 2025 → Q1 2026]
תשואה YTD: -5.57% | מסלול: מחקה S&P 500
הפקדות ברבעון: ₪7,822 | דמי ניהול: ₪298
```
